# Figurium

Курсовой проект интернет-магазина фигурок на `Next.js + React + Tailwind CSS + PostgreSQL`.

## Что уже есть

- главная страница с несколькими блоками: каталог, о нас, преимущества, CTA, контакты
- каталог с поиском, фильтрацией и сортировкой
- отдельные карточки товаров
- интерактивный 3D-просмотр каждой фигурки через `react-three-fiber`
- корзина с добавлением и оформлением демонстрационного заказа
- профиль, регистрация и вход
- база данных PostgreSQL для пользователей
- папка `public/catalog` с готовыми изображениями товаров

## Настройка PostgreSQL

1. Создайте локальную базу, например `figurium`
2. Откройте файл `.env`
3. Укажите строку подключения в формате:

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/figurium"
JWT_SECRET="figurium-demo-secret-please-change"
```

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Инициализировать таблицы:

```bash
npm run db:init
```

3. Создать демо-пользователя:

```bash
npm run db:seed
```

4. Запустить проект:

```bash
npm run dev
```
